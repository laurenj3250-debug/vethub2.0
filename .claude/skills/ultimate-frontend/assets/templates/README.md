# Frontend Templates

This directory contains starter templates for common frontend projects.

## Available Templates

### 1. landing-page
Modern SaaS landing page with:
- Hero section with CTA
- Features grid
- Testimonials
- Pricing section
- FAQ
- Footer

**Tech:** HTML, Tailwind CSS, vanilla JavaScript

### 2. react-dashboard
Admin dashboard template with:
- Sidebar navigation
- Data tables
- Charts (Chart.js)
- Form components
- Modal dialogs

**Tech:** React, Tailwind CSS, shadcn/ui, React Router

### 3. portfolio
Personal portfolio site with:
- About section
- Projects showcase (bento grid)
- Skills/tech stack
- Contact form

**Tech:** HTML, Tailwind CSS, minimal JavaScript

### 4. react-app-shell
Full-featured React application boilerplate with:
- Routing (React Router)
- State management (Zustand)
- Form handling (React Hook Form + Zod)
- UI components (shadcn/ui)
- Dark mode
- Authentication layout

**Tech:** React, TypeScript, Tailwind CSS, Vite

## Usage

Copy the desired template folder to your project directory and customize as needed. Each template includes a README with specific setup instructions.
